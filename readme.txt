A chart creation block build upon Chart.js

== Description ==

Create custom charts by filling the block form.

Chose between bar, line, pie, and other types of charts!

Give your data personalized colors!

Easily define your data labels!

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/charts` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
